# htmlBlock

html 片段代码生成工程 

```
 nodejs + express + hbs + less
 
```

1、安装包
```
npm install bower -g

npm install

bower install

```

## 运行
```
npm start
```
访问
```
 http://127.0.0.1
```

## 目录结构
```
  block      //html片段源码
   |----
      |--  *.html
      |--  *.less
   |---
   
  lib   // hbs to html   less to css
```
